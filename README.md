# 答辩讲稿打磨工具 (thesis-to-defense-script)

把博士或硕士学位论文打磨成正式答辩讲稿的 WorkBuddy Skill。

## 功能介绍

- 从论文绪论提炼层层递进的背景逻辑，压缩到 6-9 页讲稿
- 把结果章节改写成环环相扣的叙事，每页按"承上 + 方法/结果 + 启下"三段式组织
- 按正式答辩场合调整学术语言风格，去除论文书面套语和行业黑话
- 把局限性从正文中搬出，集中整理成"可继续推进的方向"
- 内置六轮迭代节奏（结构→背景→结果→语言→局限→配速），每轮聚焦一类问题
- 附检查清单和常见反模式速查，定稿前逐项核对

## 适用场景

- 学位论文已定稿，需要撰写 30-45 分钟答辩用的逐页讲稿
- 听众是答辩委员会专家 —— 懂领域但不熟悉论文细节
- 需要把书面论文转化为口语化但仍正式的口头汇报

## 安装

将此仓库克隆到 WorkBuddy 的 skills 目录：

```bash
git clone https://github.com/yingsongbei/dabianjianggao.git ~/.workbuddy/skills/thesis-to-defense-script
```

或直接复制 `SKILL.md` 到 `~/.workbuddy/skills/thesis-to-defense-script/SKILL.md`。

## 使用

在 WorkBuddy 中提到"答辩讲稿""打磨讲稿""逐页讲稿""答辩 PPT 讲稿"等关键词即可触发。也可手动调用：

```
/skill thesis-to-defense-script
```

## 触发关键词

答辩讲稿、答辩 PPT 讲稿、逐页讲稿、答辩报告、汇报口播、40 分钟版讲稿、修改讲稿、打磨讲稿

## 许可

仅供学术用途参考，使用者对讲稿内容的学术准确性负责。
